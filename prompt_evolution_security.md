# ⚔️ 資安模式 (Security Mode) 提示詞演進史 (v0.0.1 -> v0.0.15)

本文件旨在以「檔案比較」的形式，精確追蹤 `Daily_Cyber_Fin` 專案中，針對三種核心 AI 輸出：**Telegram 新聞**、**Telegram 教學**、**Markdown 日報** 的提示詞 (Prompt) 的完整演變過程。

---

## 1. 📢 資安 TG 新聞 (Security Telegram News)

此提示詞的目標是生成每日發送到 Telegram 的「資安新聞懶人包」，追求**易讀性**與**即時性**。

### v0.0.1
> 最初版本，僅要求整理和使用繁中。
```
你是資安情報官。請整理以下情報。
【內容】{raw}
【格式要求】
1. **必須使用繁體中文 (Traditional Chinese)**。
2. 每個新聞請列出：
   - 標題 (Emoji: 🚨, 🛡️)
   - 📅 **事件背景**：(一句話解釋發生什麼事)
   - 🔗 **原始連結**：(必須附上)
3. 嚴禁使用 Markdown 符號 (如 ** 或 ##)，請用 Emoji 排版。
```

### v0.0.3
> 嘗試將新聞與教學合併在一個 Prompt，但很快被放棄。此為過渡版本。

【內容】{raw}
【格式要求】

2. 新聞列表：`[MM/DD]` 標題 (Emoji) + 連結。
3. CISSP 微課程：針對最重要的一則新聞，進行技術解讀 (案例/紅隊攻擊/藍隊防禦)。
4. **嚴禁 Markdown 粗體** (Telegram 手機版閱讀用)。
```

【新聞】{raw}

1. **全篇必須使用繁體中文**。

   (深入技術細節：駭客利用了什麼底層機制？Payload 是什麼概念？)

## 3. 📄 資安 MD 日報 (Security Markdown Log)

此提示詞的目標是生成一份**極度詳盡、結構完整**的 Markdown 報告，專門用於訓練 AI 知識庫 (如 Google NotebookLM)。

### v0.0.1
> 最初版本，幾乎沒有任何指導。
```

{raw}
```

### v0.0.3

```

【輸入資料】
{raw}

標題：# 🛡️ 資安戰情日報 ({today_str})

## 1. 🌍 全球威脅快報 (News Briefing)
- 請列出所有新聞，並附上 `[MM/DD]` 日期與連結。
- 每則新聞請多寫一行「影響評估」。

## 2. 🎓 CISSP 深度戰術分析 (Deep Dive)
請挑選今日最具指標性的攻擊事件，進行深度剖析：
- **📚 案例背景**：發生了什麼？

- **⚔️ 紅隊視角 (Red Team)**：駭客使用了什麼 CVE？Payload 原理？(請詳細說明技術細節)
